# Welcome to Brand Analytics

Brand Analytics is a web application built in the Ruby programming language using the Rails framework. Brand Analytics uses the following technologies: (1) Ruby, (2) Rails, (3) JavaScript, (4) jQuery, (5) underscore.js, and (6) Twitter API. Brand Analytics is an app that allows users to analyze a body of text, either through uploaded textual data or JSON responses from the Twitter API, for analyzing the general sentiment of the text.
